I have created an interactive Hospital Dashboard that presents key patient and operational insights in a clear and visually appealing format. It includes important summary metrics such as total patients, average patient time, patient satisfaction score, number of departments, and case manager count, giving a quick overview of hospital performance.

The dashboard also features visualizations that analyze patient distribution by department referral, gender, and month, helping to identify trends and patterns over time. In addition, it highlights patient satisfaction scores across different departments, allowing for performance comparison. A detailed table is included to provide patient-level information for deeper analysis.

Overall, the dashboard effectively transforms raw data into meaningful insights, making it easier to monitor performance and support data-driven decision-making in a healthcare environment.
